# Reproducing Results

## Quick Start

```bash
git clone https://github.com/wukao1985/post-scarcity-abm.git
cd post-scarcity-abm
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

Tested with Python 3.11+ (developed on 3.14.2).

## Reproducing All Sweeps

Each sweep can be run independently. Total wall-clock time: ~2-3 hours on a modern laptop.

```bash
# Sweep 1: V2 Replication (2,250 runs, ~15 min)
python models/pathway_a_abm/runner.py 1
# → data/simulation/sweep1_results.csv

# Sweep 2: Automation Speed (3,000 runs, ~20 min)
python models/pathway_a_abm/runner.py 2
# → data/simulation/sweep2_automation_speed.csv

# Sweep 3: Virtual World Quality (3,600 runs, ~25 min)
python models/pathway_a_abm/runner.py 3
# → data/simulation/sweep3_virtual_world.csv

# Sweep 4: Collectivism Index (3,600 runs, ~25 min)
python models/pathway_a_abm/runner.py 4
# → data/simulation/sweep4_collectivism.csv

# Sweep 5: Archetype Time Series (100 runs, ~5 min)
python models/pathway_a_abm/runner.py 5
# → data/simulation/sweep5_archetypes.csv

# Sweep 6: Full 10-Scenario Grid (4,500 runs, ~30 min)
python models/pathway_a_abm/runner.py 6
# → data/simulation/sweep6_full_grid.csv
```

Or run all at once:

```bash
make all-sweeps
```

## Reproducing Figures

After sweeps complete, generate all figures:

```bash
python scripts/sweep2_figures.py   # → reports/figures/sweep2_*.png
python scripts/sweep3_figures.py   # → reports/figures/sweep3_*.png
python scripts/sweep4_figures.py   # → reports/figures/sweep4_*.png
python scripts/sweep5_figures.py   # → reports/figures/sweep5_*.png
python scripts/sweep6_figures.py   # → reports/figures/sweep6_*.png
```

Or:

```bash
make figures
```

## Running Tests

```bash
python -m pytest tests/ -v
```

## Using the Makefile

```bash
make sweep1          # Run sweep 1 only
make sweep2          # Run sweep 2 only
make all-sweeps      # Run all 6 sweeps sequentially
make figures         # Generate all figures from existing data
make test            # Run test suite
make clean-data      # Remove simulation output CSVs
```

## Reproducibility Notes

- All simulations use deterministic seeding via `seed` parameter
- Results should be identical across runs on the same platform
- Minor floating-point differences may occur across Python/NumPy versions
- Each sweep's runner prints progress to stdout (e.g., "100/3600 runs complete")
- Sweep 1 figures are generated by `models/pathway_a_abm/report.py`
